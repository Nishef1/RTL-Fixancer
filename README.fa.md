# RTL Fixancer

RTL Fixancer یک افزونه خصوصی و دامنه‌محور برای Chrome است که نمایش متن فارسی، عربی و عبری را بدون ارسال محتوای صفحه به بیرون از مرورگر بهبود می‌دهد.

## تغییرات نسخه ۴

نسخه ۴ معماری قدیمی و اجرای دائمی روی همه سایت‌ها را حذف کرده است:

- دیگر content script سراسری با `<all_urls>` وجود ندارد.
- افزونه هنگام نصب به همه سایت‌ها دسترسی دائمی نمی‌گیرد.
- دسترسی فقط هنگام فعال‌کردن یک سایت و با تأیید کاربر درخواست می‌شود.
- اسکریپت فقط برای دامنه‌های فعال به‌صورت پویا ثبت می‌شود.
- فقط frame اصلی صفحه پردازش می‌شود.
- یک `MutationObserver` رویدادمحور جایگزین pollingها و timerهای دائمی شده است.
- مقدار اصلی تمام attributeهای تغییرکرده ذخیره و هنگام غیرفعال‌سازی دقیقاً بازگردانده می‌شود.
- فایل‌های patch و monkey-patchهای قدیمی حذف شده‌اند.
- خروجی PDF از مسیر استاندارد Print / Save as PDF مرورگر استفاده می‌کند و دیگر صفحه‌های بی‌نهایت را اسکرول نمی‌کند.

## قابلیت‌ها

- تشخیص فارسی، عربی و عبری.
- مدیریت امن متن‌های ترکیبی RTL و LTR.
- فونت وزیر، شبنم یا فونت اصلی سایت.
- تنظیم اندازه فونت و حساسیت تشخیص.
- مدیریت دسترسی و فعال‌سازی جداگانه برای هر سایت.
- سازگاری هدفمند با ChatGPT، Gemini، Google AI Studio، Perplexity و DeepSeek.
- حفظ جهت چپ‌به‌راست برای کدها و کنترل‌های رابط سایت.
- رابط افزونه به زبان انگلیسی و فارسی.

## نصب توسعه‌ای

۱. مخزن را دانلود یا clone کن.  
۲. وارد `chrome://extensions` شو.  
۳. **Developer mode** را فعال کن.  
۴. روی **Load unpacked** بزن و ریشه مخزن را انتخاب کن.  
۵. یک سایت عادی را باز کن، روی افزونه بزن و همان سایت را فعال کن.

حداقل نسخه موردنیاز Chrome 120 است.

## بررسی پروژه

با Node.js 22 یا جدیدتر:

```bash
npm run check
```

این دستور manifest، مجوزها، ساختار امنیتی و تست‌های هسته را بررسی می‌کند. تغییرات runtime باید علاوه بر تست خودکار، داخل Chrome نیز روی سایت عمومی و سرویس‌های چت پشتیبانی‌شده آزمایش شوند.

## حریم خصوصی

افزونه کاملاً محلی کار می‌کند و شامل analytics، کد راه‌دور، API شبکه یا ارسال محتوای صفحه نیست. دسترسی هر دامنه فقط با انتخاب و تأیید خود کاربر داده می‌شود.
